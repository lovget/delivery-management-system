# Delivery Management System

## Описание проекта
Delivery Management System — REST-приложение для управления заказами и процессом доставки.

Проект реализован на Spring Boot и построен по архитектуре:
Controller → Service → Repository.

## Функциональность (Лабораторная работа №1)

- Получение списка заказов
- Получение заказа по ID
- Фильтрация заказов по статусу
- Создание нового заказа
- Удаление заказа
- Использование DTO
- Разделение слоёв приложения
- Возврат корректных HTTP-статусов (200, 404, 400)

## Используемые технологии

- Java
- Spring Boot
- Maven
- Git
- SonarCloud
- Checkstyle

## Архитектура

- Controller — обработка HTTP-запросов
- Service — бизнес-логика
- Repository — работа с данными
