# Basic REST Service
Delivery Management System

## 1. Название темы

Создание Spring Boot приложения с реализацией REST API для одной ключевой сущности предметной области.

## 2. Выполняемые функции

В рамках работы реализовано:

1. Создано Spring Boot приложение.
2. Реализован REST API для сущности Order.
3. Реализован GET endpoint с использованием @PathVariable.
4. Реализован GET endpoint с использованием @RequestParam.
5. Реализована слоистая архитектура:
   - Controller — обработка HTTP-запросов;
   - Service — бизнес-логика;
   - Repository — работа с данными.
6. Используется DTO для передачи данных в API-ответе.
7. Подключён анализатор кода SonarCloud.

## 3. Описание проекта

Проект представляет собой REST-сервис управления заказами в системе доставки.

Основной сущностью является Order (заказ).
Сервис позволяет получать список заказов, получать заказ по идентификатору и выполнять поиск заказов по статусу.

Проект реализован с использованием многослойной архитектуры и соответствует принципам объектно-ориентированного программирования.
